# Glossary

| Term | Definition |
|------|-----------|
| **ACP** | Agent Client Protocol — IDE integration bridge |
| **A2UI** | Agent-to-UI protocol for canvas rendering |
| **Allowlist** | Approved sender list for DM access |
| **Auto-reply** | Response generation engine (`src/auto-reply/`) |
| **Bootstrap** | First-run initialization sequence |
| **Channel adapter** | Plugin implementing messaging platform integration |
| **ClawHub** | OpenClaw's skill marketplace |
| **Compaction** | Context window management when approaching token limits |
| **DM pairing** | Security mechanism requiring code verification for new DM senders |
| **Doctor** | Diagnostic tool for configuration and health issues |
| **Exec approval** | User confirmation required for dangerous tool executions |
| **Gateway** | Central WebSocket server / control plane |
| **Hono** | Lightweight web framework (14KB) |
| **Jiti** | Dynamic ESM import loader used by plugin system |
| **MCP** | Model Context Protocol — external tool integration |
| **Nexus** | Working name for the clean-room rebuild |
| **Node** | Remote compute device (iOS/Android/macOS companion) |
| **OpenClaw** | The product being analyzed |
| **Pi** | OpenClaw's embedded agent runtime (`@mariozechner/pi-*`) |
| **Plugin SDK** | API surface for extensions (`openclaw/plugin-sdk/*`) |
| **QMD** | Queryable Memory Document — OpenClaw's memory format |
| **Session** | Conversation context instance (per-channel, per-peer) |
| **Skill** | Bundled AI capability (SKILL.md + frontmatter) |
| **Swabble** | Vendored Swift library for voice wake |
| **WAL** | Write-Ahead Logging — SQLite concurrency mode |
